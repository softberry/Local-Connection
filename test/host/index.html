Local LocalConnection 2.0

This is the main browser window, which contains 3 iframes. Each iframe contains another iframe.

Parent window (this one) served on http://localhost:3300 and all embedded iframes served on http://localhost:3200, so to test foreign documents connects properly.

Each embedded iframe (left.html, top.html,right.html) needs to connect each other. After a successful connection, left.html modifies top.html, top.html modifies right.html and last right.html modifies left.html.

Each page has a red ball inside with the class name 'status'. After a succesfull connection it gets modified and becomes dditional class name 'connected'. So if all red ball turns in to green, it means that connection has been established properly.
